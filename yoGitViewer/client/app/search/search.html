Search Screen

Finding a User

Enter the name of a Github user in the navbar and click the Find   button. On the user and repo screens, when a list of follows/following, or contributors comes up, you can click on a button to select that user.

Finding a Repository

Enter the name of a user, and a repository owned by that user, and click the Find   button. On the user screen, where the repositories are listed, you can click on the Repo link button to pull up the repository info.
